# Synchronize time using other network peers

## timedatectl
Current Server date

* Enable/Disable time sync: `sudo timedatectl set-ntp on/off`
* Get Timezones: `timedatectl list-timezones`
* Set Timezone: `sudo timedatectl set-timezone TIMEZONE`
